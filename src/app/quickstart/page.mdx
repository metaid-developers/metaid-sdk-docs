## Installation

Choose your preferred JavaScript package manager to install the MetaID SDK package.

<CodeGroup>

```bash {{ title: 'NPM' }}
npm install @metaid/metaid
```

```bash {{ title: 'Yarn' }}
yarn add @metaid/metaid
```

```bash {{ title: 'PNPM' }}
pnpm add @metaid/metaid
```

```bash {{ title: 'Bun' }}
bun install @metaid/metaid
```
